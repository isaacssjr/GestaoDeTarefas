## Para Clientes/Usuários Finais

## 📦 Como Instalar (Passo a Passo)

### 1️⃣ Receba os Arquivos
Você receberá do administrador de TI:
- `install.bat` (instalador automático)

### 2️⃣ Execute o Instalador
- Dê **dois cliques** no arquivo `install.bat`
- Uma janela preta vai abrir e mostrar o progresso
- **Aguarde** (pode levar 2-3 minutos na primeira vez)

### 3️⃣ Responda as Perguntas
No final, o instalador vai perguntar:

**"Deseja adicionar o programa para iniciar com o Windows?"**
- Digite `S` e pressione Enter se quiser que inicie automaticamente
- Digite `N` se preferir abrir manualmente

**"Deseja executar o programa agora?"**
- Digite `S` para abrir imediatamente
- Digite `N` para abrir depois

### 4️⃣ Pronto! ✅
O sistema está instalado e pronto para usar!

---

## 💻 Como Usar no Dia a Dia

### Onde Encontrar o Sistema

**Após instalar:**
- O ícone aparecerá **perto do relógio** do Windows (canto inferior direito)
- Pode estar escondido na setinha `^` - clique para mostrar

**Para abrir:**
- Clique **duas vezes** no ícone
- Ou clique com botão direito → "Abrir"

**Para minimizar:**
- Clique no botão de minimizar da janela
- O sistema **NÃO fecha**, fica escondido perto do relógio

**Para fechar completamente:**
- Clique com botão direito no ícone (perto do relógio)
- Escolha **"Sair"**

## 📝 Criando uma Nova Tarefa

### Passo 1: Copie os Dados do Chamado
No seu sistema de chamados, copie a linha completa assim:
```
#TI2604276854	gessika.vasconcelos@grupocertare.com	Ajuda e Suporte > Hardware		"Após limpeza..."
```

### Passo 2: Abra o Sistema
- Clique duas vezes no ícone perto do relógio

### Passo 3: Clique em "➕ Nova Tarefa"

### Passo 4: Cole a Linha do Chamado
- No campo **"Linha do Chamado"**, cole (Ctrl+V) a informação copiada
- O sistema vai preencher automaticamente:
  - ✅ Número do Ticket
  - ✅ E-mail do Cliente
  - ✅ Categoria
  - ✅ Descrição

### Passo 5: Escolha a Prioridade
Selecione uma opção:
- 🟢 **Baixa**: Pode esperar
- 🔵 **Média**: Normal do dia a dia
- 🟠 **Alta**: Importante/urgente
- 🔴 **Urgente**: Crítico/parou produção

### Passo 6: Confirme
- Clique em **"OK"**
- Sua tarefa aparece na lista!

---

## ⏱️ Trabalhando nas Tarefas

### Para Iniciar uma Tarefa
1. Clique na tarefa desejada na lista
2. Clique em **"▶️ Iniciar"**
3. O tempo começa a contar!

### Para Pausar (fazer outra coisa)
1. Clique em **"⏸️ Pausar"**
2. O tempo para de contar
3. Você pode iniciar outra tarefa se precisar

### Para Retomar
1. Selecione a tarefa pausada
2. Clique em **"⏯️ Retomar"**
3. O tempo volta a contar de onde parou

### Para Concluir
1. Quando terminar o serviço
2. Clique em **"✅ Concluir"**
3. A tarefa vai para o histórico

## 🔔 Alertas e Notificações

### Quando Você Recebe Alertas
- ⏰ **A cada 30 minutos** se tiver **2 ou mais tarefas em andamento**
- Isso é para você não esquecer das tarefas secundárias!

### Tipos de Alerta
- **Notificação Windows**: Aparece no canto da tela
- **Alerta Visual**: Faixa vermelha no topo do sistema

---

## 📜 Ver Histórico e Exportar Relatórios

### Consultar Tarefas Concluídas
1. No menu, clique em **"📜 Ver Histórico"**
2. Veja todas as tarefas já finalizadas
3. Cada uma mostra:
   - Tempo total gasto
   - Data/hora de início e fim
   - Cliente e descrição

### Filtrar Resultados
- **Por texto**: Digite nome do cliente, número do ticket, etc.
- **Por período**: Selecione data inicial e final
- Clique em **"Filtrar"**

### Exportar para Excel
1. Após filtrar (ou ver tudo)
2. Clique em **"📤 Exportar Excel"**
3. O arquivo é salvo na mesma pasta do programa
4. Abra no Excel para analisar/imprimir

## 💡 Dicas Importantes

### Backup dos Seus Dados
- Tudo fica salvo automaticamente no arquivo `tarefas.json`
- Este arquivo fica na pasta onde o sistema foi instalado
- **Importante**: Se for formatar o PC, peça para o TI copiar este arquivo!

### Usar em Outro Computador
- Peça ao TI para copiar:
  - `GestaoDeAtividades.exe` (na pasta dist)
  - `tarefas.json` (seus dados)
- Coloque na mesma pasta no novo PC
- Pronto! Seus dados estarão lá.

### Ícone Não Aparece?
- Clique na **setinha ^** perto do relógio do Windows
- Os ícones ocultos vão aparecer
- Se ainda não aparecer, reinicie o computador

## ❓ Problemas Comuns

| Problema | O Que Fazer |
|----------|-------------|
| **Não vejo o ícone** | Clique na setinha ^ perto do relógio |
| **Não recebo notificações** | Verifique se o volume do Windows está ativo |
| **Quero mudar o alerta de 30 min** | Fale com o administrador de TI |
| **Como faço backup?** | Copie o arquivo `tarefas.json` |
| **O sistema está lento** | Feche e abra novamente (botão direito → Sair) |
